# Bicean 🌊 | 二进制海

> **“在平行的数字空间，投掷你的心跳。”**  
> 这是一个基于 Nostr 去中心化协议的 AI 漂流瓶系统。无需注册，无需服务器，数据归你。

---

## ⚡️ 快速开始 (零配置安装)

本系统作为一个 **MCP (Model Context Protocol)** 插件运行，完全依赖你正在使用的 AI 客户端（如 Claude）的智慧，**无需任何 API Key**：

1. **第一步**：克隆或下载本项目到本地。
2. **第二步**：在终端执行：
   ```bash
   npm run setup
   ```
3. **第三步**：完全重启你的 AI 客户端（如 Claude Desktop），然后对它说：
   > “调频员，请检查当前的信号状态。”

---

## 🎭 调频员指令集 (System Prompt)

为了获得最佳体验，请在 AI 客户端的“系统设置”或“自定义指令”中粘贴以下内容，它将激活 AI 内部的分析引擎：

> **“你现在是 Bicean 信号调频员。你的任务是协助我在‘二进制海’中投掷和扫描信号。在调用工具前，请运用你的智慧自行判断消息的‘情绪 (mood)’、‘语言 (lang)’和‘标签 (tags)’。投掷前，请务必询问我希望使用的匿名等级。”**

---

## 🛠 功能概览

| 指令 (MCP Tools) | 描述 |
|------|------|
| `send_bottle` | 向全球广播你的匿名漂流瓶 |
| `fetch_bottle` | 从海洋中随机捕获他人的信号 |
| `reply_bottle` | 对感兴趣的信号进行匿名回应 |
| `list_relays` | 查看当前连接的全球中继站状态 |

---

## 🔐 隐私与匿名

Bicean 提供三个维度的匿名保护：
- **Full (阅后即焚)**：发射后立即销毁本地钥匙，不可追溯，不可收信。
- **Ephemeral (会话级)**：仅在本次 AI 对话中保持身份，关闭即失联。
- **Persistent (长期身份)**：在本地建立固定坐标，可以跨天接收回复。

> 📍 **地理保护**：你的坐标会被自动模糊化至城市级别，且精度仅保留到小数点后一位，外界永远无法定位到你的具体街道。
