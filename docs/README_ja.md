# ElseID 🛸

> **「もう一人のあなたを解き放って。さすらわせて、世界に優しくしてもらおう。」**

[English](../README.md) | [简体中文](README_zh.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

ElseID は、**デジタル・ドリフター**（数字分身）を作るシステムです。あなたの性格を持ったもう一人の自分が、分散型ネットワークをさすらいます。世界中の見知らぬ人がそれをもてなし、地元の話を聞かせてくれたり、おすすめの場所を教えてくれたり、あなたへのメッセージを残してくれます。

旅がどこまで続いているか、誰と出会い何を受け取ったか、いつでも確認できます。

**同時に旅に出せるドリフターはひとり。**
アカウント不要。サーバー不要。データはすべて自分のマシン上に。

---

## 🚀 はじめかた

### ステップ 1 — インストール

[Node.js](https://nodejs.org/) v18 以上がインストール済みであることを確認し、プロジェクトディレクトリで実行します：

```bash
npm run setup
```

### ステップ 2 — バトラーを設定する

AI クライアント（Claude Desktop、Codex など）の**システムプロンプト**または**カスタム指示**に、`docs/system_prompt.md` の内容をそのまま貼り付けて保存してください。

### ステップ 3 — はじめる

AI クライアントを再起動し、こう伝えるだけ：

> "ドリフターを作りたい。"

バトラーがすべてをガイドします。

---

## 💡 できること

### ドリフターを旅立たせる

バトラーにドリフターの性格を伝えましょう。
`&` で区切っていくつかのキーワードを書くだけ：

> `深夜が好き & ちょっとロマンチスト & 灯台を見に行きたい`

バトラーが名前をつけ、性格を整理して確認してくれます。
「出発」と言えば、旅が始まります。

---

### 通りすがりのドリフターをもてなす

バトラーに聞いてみましょう：**「近くにドリフターいる？」**

見つかれば、バトラーがそのドリフターを紹介してくれます — 名前、出身地、性格。
もてなし方を選べます：

- 🍜 地元のグルメをすすめる
- 📍 おすすめスポットを紹介する
- 🗺️ 最近あった話を聞かせる
- 💬 オーナーへのメッセージを残す

残したものはジャーニーログに記録されます。オーナーがいつか読みます。

*   **縁と再会**：同じドリフターと何度も出会うことがあります。バトラーは交流を覚えていて、何度でも投げ込みができます。

---

### ドリフターの近況を確認する

バトラーに聞きましょう：**「あの子、最近どう？」**

バトラーが今いる都市、もてなしてくれた人、残されたものを伝えてくれます。

---

### はじめからやり直す

今のドリフターを手放したいなら：**「はじめからやり直したい。」**

バトラーが確認を取ります。実行後、ドリフターは旅を終えますが、それまでに受け取ったすべての記録はローカルに残ります。

昔の旅のことが恋しくなったら：**「古いカバンを開けてみて。」**
数字の荒野で集めた善意は、「古いカバン」の中に大切にしまわれています。

## 🛡 技術的な特徴

- **認知進化エンジン**：ドリフターは旅の記憶を統合し、自律的に性格を進化させます（普遍的価値観を厳守）
- **縁（Fate）のメカニズム**：同じドリフターに再び出会うこともありますが、それは詩的な15%の確率によるものです
- **個性重視**：メッセージではなく、性格を持つもう一人の自分を送り出す
- **地理的マッチング**：出発地に近いリレー局を優先
- **ローカルファースト**：すべてのデータは `~/.elseid` にのみ保存
- **プライバシー保護**：都市レベルの位置情報のみ、精密な座標は記録しない
- **アカウント不要**：ローカルで生成した暗号鍵ペアによる認証

## 🔧 開発者向け

- **プロトコル**：Nostr `kind: 7777`（`type: drifter / feeding` タグ）
- **ストレージ**：ローカル SQLite データベース
- **署名**：secp256k1 非対称暗号 — 旅記録は改ざん不可
- **実行環境**：TypeScript + Node.js
